# "DATA TYPES" IN MySQL

## NEUMERIC Data Types
1. INT / INTEGER (8 Bytes)
2. TINYINT (1 Byte)
3. SMALLINT (2 Byte)
4. FLOAT (4 Bytes) 
5. DOUBLE (8 Bytes)
6. DECIMAL (M, D)
7. MEDIUMINT (4 Bytes)
8. BIGINT (16 / 32 / 64 Bytes)


## TEXT
1. CHAR (1 Byte)
2. VARCHAR (Fixed Length)
3. TEXT (Variably Large)


## OTHER
1. SET (Multiple Options)
2. ENUME (Single Option)
